Some dotfiles and two git hooks that will link all dotfiles into your home directory on commit or merge/pull.

## Installation (Git Read-Only)

```
git clone git@github.com:tralafiti/dotfiles.git ~/.dotfiles
~/.dotfiles/init.sh
```
